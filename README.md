
# A PyGame Game

A python game made using pygame library.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Mikepool117/Another-PyGame-Game
```


Install dependencies

```bash
  pip install pygame
```

Start the program

```bash
  python main.py
```
